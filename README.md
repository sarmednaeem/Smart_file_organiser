# 🗂️ Smart File Organizer

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

A powerful Python automation tool that intelligently organizes files on your computer with a modern GUI, real-time monitoring, and duplicate detection.

![Smart File Organizer Demo](assets/demo.gif)

## ✨ Features

- **🔍 Smart Scanning** - Scan any folder (Downloads, Desktop, Documents, etc.)
- **📁 Intelligent Sorting** - Organize files by type, date, or custom rules
- **✏️ Smart Renaming** - Clean up messy filenames automatically
- **👀 Watch Mode** - Real-time auto-organization as files are added
- **🔄 Duplicate Detection** - Find and handle duplicate files using content hashing
- **🖥️ Modern GUI** - User-friendly interface built with CustomTkinter
- **⚙️ Customizable** - Configure file categories and organization rules
- **↩️ Undo Support** - Revert recent organization actions
